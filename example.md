# Markdown to HTML Converter

Welcome to the **md2html** tool demonstration!

## Features

This tool converts Markdown files to beautiful HTML documents with:

- **Embedded images** as base64 data URIs
- **Google Fonts** integration (default: Open Sans)
- **Responsive design** that works on all devices
- **Clean, modern styling**
